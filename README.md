# inecuaciones
De las inecuaciones a la programación lineal
Recurso interactivo para **1º de Bachillerato (Matemáticas Generales)**: repaso guiado de **inecuaciones**, **sistemas** y **programación lineal**, con ejemplos contextualizados en familias de **Formación Profesional** del centro.

**Proyecto marco:** CITE-STEAM **“IA, FP y robótica inclusiva”**  
**Centro:** IES San José (Badajoz)  
**Coordinación:** Cristina Ramos García.

## Qué incluye
- Guía breve y visual de técnicas: 1D (lineales, cuadráticas, racionales), 2D (semiplanos y sistemas) y PL (vértices/óptimo).
- Ejemplos contextualizados en FP: Peluquería y Estética, Madera y Mueble, Soldadura, Automoción, Industria Alimentaria, PRL y Administración.
- Actividades de práctica con **autocorrección** (respuesta múltiple/decisión y respuestas guiadas).
- Sección de apoyo para **GeoGebra** (sintaxis de restricciones y sistemas).
- Modo impresión/“ficha” (según versión publicada).
